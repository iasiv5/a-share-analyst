# 基本面指标说明

## 估值指标

### PE (市盈率)
```
PE = 股价 / 每股收益(EPS)
PE(TTM) = 当前市值 / 最近四个季度净利润
```
- PE < 15：低估值
- 15 < PE < 25：合理估值
- PE > 25：高估值
- 注意：亏损股票PE无意义，需结合行业对比

### PB (市净率)
```
PB = 股价 / 每股净资产
```
- PB < 1：破净股，可能低估或资产质量差
- 1 < PB < 3：合理区间
- PB > 5：可能高估

### PS (市销率)
```
PS = 当前市值 / 年度营业收入
```
- 适用于亏损但有收入增长的公司
- PS < 1：可能低估
- PS > 10：可能高估

### PEG (市盈增长比率)
```
PEG = PE / 净利润增长率(%)
```
- PEG < 1：相对低估
- PEG = 1：合理
- PEG > 1：相对高估

## 盈利能力

### ROE (净资产收益率)
```
ROE = 净利润 / 平均股东权益 × 100%
```
- ROE > 15%：优秀
- 10% < ROE < 15%：良好
- ROE < 10%：一般

### ROA (总资产收益率)
```
ROA = 净利润 / 平均总资产 × 100%
```
- ROA > 10%：优秀
- 5% < ROA < 10%：良好

### 毛利率
```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
```
- 毛利率 > 40%：高毛利行业(医药、软件)
- 20% < 毛利率 < 40%：中等水平
- 毛利率 < 20%：低毛利行业(零售、制造)

### 净利率
```
净利率 = 净利润 / 营业收入 × 100%
```
- 净利率 > 15%：优秀
- 5% < 净利率 < 15%：良好

## 成长能力

### 营收增长率
```
营收增长率 = (本期营收 - 上期营收) / 上期营收 × 100%
```
- 增速 > 30%：高成长
- 10% < 增速 < 30%：中等成长
- 增速 < 10%：稳定型

### 净利润增长率
```
净利增长率 = (本期净利 - 上期净利) / 上期净利 × 100%
```
- 持续稳定增长优于大起大落

### 扣非净利润增长率
- 排除非经常性损益的净利润增长
- 更能反映主营业务真实增长

## 财务健康

### 资产负债率
```
资产负债率 = 总负债 / 总资产 × 100%
```
- < 40%：财务稳健
- 40% - 60%：正常水平
- > 70%：财务风险较高
- 注意：银行等金融行业除外

### 流动比率
```
流动比率 = 流动资产 / 流动负债
```
- > 2：短期偿债能力强
- 1 - 2：正常
- < 1：短期偿债压力大

### 速动比率
```
速动比率 = (流动资产 - 存货) / 流动负债
```
- > 1：偿债能力良好

### 经营现金流/净利润
```
现金流比率 = 经营活动现金流净额 / 净利润
```
- > 1：利润质量高，现金回收好
- < 0.5：利润质量存疑

## 运营效率

### 存货周转率
```
存货周转率 = 营业成本 / 平均存货
存货周转天数 = 365 / 存货周转率
```
- 周转率越高，运营效率越好

### 应收账款周转率
```
应收周转率 = 营业收入 / 平均应收账款
应收周转天数 = 365 / 应收周转率
```
- 周转天数短：回款快，坏账风险低

### 总资产周转率
```
总资产周转率 = 营业收入 / 平均总资产
```
- 反映资产利用效率

## 分红指标

### 股息率
```
股息率 = 每股分红 / 股价 × 100%
```
- 股息率 > 3%：高股息股票
- 长期稳定分红表明财务健康

### 分红率
```
分红率 = 每股分红 / 每股收益 × 100%
```
- 合理分红率：30%-50%

## 综合评分模型

| 维度 | 指标 | 权重 | 优秀 | 良好 | 一般 |
|------|------|------|------|------|------|
| 估值 | PE | 20% | <15 | 15-25 | >25 |
| 估值 | PB | 10% | <2 | 2-4 | >4 |
| 盈利 | ROE | 20% | >15% | 10-15% | <10% |
| 盈利 | 净利率 | 10% | >15% | 5-15% | <5% |
| 成长 | 营收增速 | 15% | >30% | 10-30% | <10% |
| 成长 | 净利增速 | 15% | >30% | 10-30% | <10% |
| 安全 | 资产负债率 | 10% | <40% | 40-60% | >60% |
