# 基本面指标说明

## 估值指标

### PE (市盈率)
- PE < 15：低估值
- 15 < PE < 25：合理估值
- PE > 25：高估值
- 注意：亏损股票PE无意义，需结合行业对比和PE TTM（最近四个季度）

### PB (市净率)
- PB < 1：破净股，可能低估或资产质量差
- 1 < PB < 3：合理区间
- PB > 5：可能高估
- 银行、地产等行业常用 PB 估值

### PS (市销率)
- 适用于亏损但有收入增长的公司（如 SaaS、互联网）
- PS < 1：可能低估
- PS > 10：可能高估
- 高成长行业可接受更高 PS

### PEG (市盈增长比率)
- PEG < 1：相对低估（成长性好）
- PEG = 1：合理估值
- PEG > 1：相对高估（或增速不及预期）

## 盈利能力

### ROE (净资产收益率)
- ROE > 15%：优秀（巴菲特标准）
- 10% < ROE < 15%：良好
- ROE < 10%：一般
- 注意：高ROE可能是杠杆驱动，需结合资产负债率分析

### ROA (总资产收益率)
- ROA > 10%：优秀
- 5% < ROA < 10%：良好
- 轻资产公司 ROA 通常更高

### 毛利率
- 毛利率 > 40%：高毛利行业（医药、软件、白酒）
- 20% < 毛利率 < 40%：中等水平
- 毛利率 < 20%：低毛利行业（零售、制造、公用事业）
- 毛利率下降可能预示竞争加剧或成本上升

### 净利率
- 净利率 > 15%：优秀
- 5% < 净利率 < 15%：良好
- 净利率波动大：可能存在非经常性损益

## 成长能力

### 营收增长率
- 增速 > 30%：高成长（可能透支未来）
- 10% < 增速 < 30%：中等成长
- 增速 < 10%：稳定型或衰退
- 连续3年以上 >20% 增长：优秀成长股

### 净利润增长率
- 持续稳定增长优于大起大落
- 净利增速 > 营收增速：盈利能力提升
- 净利增速 < 营收增速：可能面临成本压力

### 扣非净利润增长率
- 排除非经常性损益的净利润增长
- 更能反映主营业务真实增长

## 财务健康

### 资产负债率
- < 40%：财务稳健
- 40% - 60%：正常水平
- > 70%：财务风险较高
- 注意：银行、地产等行业负债率天然较高，需用行业对比

### 流动比率
- > 2：短期偿债能力强
- 1 - 2：正常
- < 1：短期偿债压力大，可能面临流动性危机

### 速动比率
- > 1：偿债能力良好（扣除存货后）
- 比流动比率更严格，适用于存货周转慢的公司

### 经营现金流/净利润
- > 1：利润质量高，现金回收好
- 0.5 - 1：正常
- < 0.5：利润质量存疑（可能存在大量应收账款）

## 运营效率

### 存货周转率
- 周转率越高，运营效率越好
- 周转天数 > 180：可能存在存货积压

### 应收账款周转率
- 周转天数短：回款快，坏账风险低
- 周转天数 > 90：可能存在应收账款积压

### 总资产周转率
- 反映资产利用效率
- 轻资产公司周转率通常更高

## 分红指标

### 股息率
- 股息率 > 3%：高股息股票（相当于债券收益）
- 长期稳定分红（5年以上）表明财务健康

### 分红率
- 合理分红率：30%-50%（保留现金再投资 vs 分红）
- 分红率 > 80%：可能缺乏成长机会
- 分红率 < 10%：可能更关注成长或现金不足

## AKShare获取基本面数据

```python
import akshare as ak

# 财务指标
df = ak.stock_financial_analysis_indicator(symbol="000001")

# 资产负债表
df = ak.stock_balance_sheet_by_report_em(symbol="000001")

# 利润表
df = ak.stock_profit_sheet_by_report_em(symbol="000001")

# 现金流量表
df = ak.stock_cash_flow_sheet_by_report_em(symbol="000001")

# 业绩预告
df = ak.stock_yjyg_em()

# 业绩快报
df = ak.stock_yjkb_em()
```

## 综合评分模型

| 维度 | 指标 | 权重 | 优秀 | 良好 | 一般 |
|------|------|------|------|------|------|
| 估值 | PE | 20% | <15 | 15-25 | >25 |
| 估值 | PB | 10% | <2 | 2-4 | >4 |
| 盈利 | ROE | 20% | >15% | 10-15% | <10% |
| 盈利 | 净利率 | 10% | >15% | 5-15% | <5% |
| 成长 | 营收增速 | 15% | >30% | 10-30% | <10% |
| 成长 | 净利增速 | 15% | >30% | 10-30% | <10% |
| 安全 | 资产负债率 | 10% | <40% | 40-60% | >60% |
